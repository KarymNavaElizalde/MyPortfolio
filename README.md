# 🛠️ DevOps Bootcamp Código Facilito: Práctica 01 - MyPortfolio

Este repositorio marca el inicio de mi formación especializada en la cultura DevOps. No es solo un contenedor de código, sino el primer nodo de un ecosistema de automatización y despliegue continuo.

### 🎯 Objetivos de la Práctica
* **AI-Driven Development:** Implementación de flujos de trabajo asistidos por IA (Copilot) para la generación de código eficiente y optimización de lógica en tiempo real.
* **Continuous Delivery (CD):** Despliegue inmediato de activos digitales en entornos productivos, reduciendo el *time-to-market* desde el primer día de desarrollo.
* **Control de Versiones Profesional:** Establecimiento de la estructura base en GitHub para el versionamiento integral y ciclo de vida de todo el software desarrollado en el bootcamp.
* **Advanced Prompting:** Aplicación de ingeniería de prompts para la resolución de problemas técnicos complejos y comunicación efectiva con modelos de lenguaje de gran escala (LLM).

### 📂 Estructura del Proyecto
* `/scripts`: Automatizaciones básicas de sistema.
* `/docs`: Documentación técnica y bitácora de aprendizaje.
* `/config`: Archivos de configuración de entorno.

### 🚀 Roadmap de Aprendizaje
- [x] Configuración de entorno local y SSH.
- [x] First Deploy & Live URL.
- [ ] Implementación de contenedores (Docker).
- [ ] Automatización de Pipelines (GitHub Actions).
